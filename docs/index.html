Experimentation Lab

A simulation engine where the true treatment effect is known by construction, used to break A/B tests the six ways they break in production and measure the damage exactly.

Three layers: a tested Python engine that simulates experiment pathologies against known ground truth; a review agent that turns a design into a decision memo grounded in those simulations; and these explainers, where every number and every animation is generated by committed, tested code. Source on GitHub.

All data on this site is synthetic, generated by the repository's simulation engine with fixed seeds. That is the point: because the true effect is set by construction, every bias and error rate shown here is measured against an answer key, which no real dataset allows. The running product examples describe a fictional short-video app.
Experimentation Lab. Educational and demonstrative; not an experimentation platform.
